# Practice 7 - PhoneBook

## Description

This project is a simple PhoneBook application using Python and PostgreSQL.

## Features

- Create PhoneBook table
- Add contact
- Import contacts from CSV
- Show contacts
- Search by name
- Search by phone prefix
- Update contact
- Delete contact

## Requirements

- Python
- PostgreSQL
- psycopg2

Install psycopg2:

pip install psycopg2-binary

Run:

python phonebook.py
